Getting Started
What is Arcanara? Getting Started

Arcanara is a tarot-focused Discord bot built for fast, readable card meanings and guided self-reflection. It supports slash commands for pulls and interpretations, and can include card images when permissions allow.

Arcanara isn't here to "tell your future." It's designed to help you think clearly, notice patterns, and explore options.

How do I see what commands are available? Getting Started

Type / in any channel where Arcanara is active and browse the command list. Most users start with /meaning for specific cards, /cardoftheday for a daily pull, or /read for a guided three-card reading.

Free & Oracle Tiers
What's included for free? Free & Oracle

Arcanara's core experience is completely free. Here's what you get:

Free Tier

  • 1 reading per day shared across /read, /threecard, and /mystery
  • /cardoftheday — daily card pull (unlimited)
  • /meaning — look up any card's meaning
  • /clarify — draw 1 clarifier card
  • /tone — 3 reading tones: poetic, quick, direct
  • /history — view your last 3 readings
  • /shuffle, /intent, /insight, /settings
  • /privacy and /forgetme for data control
What does Arcanara Oracle unlock? Oracle

Arcanara Oracle removes all limits and adds powerful new tools for deeper practice.

Oracle Tier

  • Unlimited readings — no daily cap on /read, /threecard, /mystery
  • Up to 3 clarifier cards with /clarify
  • 20 reading history entries (vs 3)
  • 5 premium tones — full, shadow, love, work, money
  • /celtic — full 10-card Celtic Cross spread
  • /journal — write, read, and search personal reflections
  • /spread — create and use custom spread layouts
  • /summary — weekly and monthly reading pattern analysis
Why can't I do another reading today? Limits

Free users get 1 reading per day shared across /read, /threecard, and /mystery. Once you've used any one of these, the others are locked until tomorrow.

/cardoftheday and /meaning are always available and don't count toward this limit.

Your daily limit resets each day. To unlock unlimited readings, upgrade to Arcanara Oracle.

How do I upgrade to Arcanara Oracle? Oracle

You can subscribe to Arcanara Oracle directly through Discord. Visit Arcanara's app page on Discord and look for the subscription option. Once subscribed, all premium features unlock instantly.

Commands
How does /meaning work? Commands

/meaning returns upright and reversed meanings for a requested card. If image permissions are enabled, Arcanara includes the card image alongside the text.

Tip: type partial card names and choose from Discord's autocomplete suggestions.

How does /cardoftheday work? Commands

/cardoftheday gives you a daily card pull for reflection. It stays consistent throughout the day so you're not re-drawing every five minutes.

This command is always free and doesn't count toward your daily reading limit.

What are reading tones? Commands

Tones control how Arcanara speaks to you. Use /tone to choose your style.

Free tones: poetic (symbolic, soft edges), quick (one clear message), direct (straight talk, no fluff).

Oracle tones: full (deep + practical), shadow (truth + integration), love (people + patterns), work (purpose + friction), money (resources + decisions).

Behavior
Are Arcanara responses always ephemeral? Behavior

Ephemeral means "only you can see it" in Discord. Most Arcanara responses are ephemeral for privacy. Some commands may respond publicly by design (for shared readings). If unsure, check the command description.

Images & Permissions
Why can't I see card images? Images

Card images require Attach Files permission in the channel. If Arcanara uses embeds, Embed Links may also be needed.

Fix: check channel permission overrides for the bot role and confirm "Attach Files" is allowed.

Which permissions does Arcanara need? Permissions

For the channel where you use Arcanara:

  • View Channel
  • Send Messages
  • Read Message History
  • Embed Links
  • Attach Files (for card images)

If commands respond but output is missing (especially images), it's almost always a channel override issue.

Privacy
Does Arcanara store my reading history? Privacy

History is opt-in only and disabled by default. Use /settings to enable or disable history tracking at any time.

What data does Arcanara store? Privacy

Arcanara stores the minimum needed to function: your preferences (tone, image settings) and any opt-in data like reading history. Use /privacy to see what's stored and /forgetme to delete everything.

Guidance
Is Arcanara "advice"? How should I use readings responsibly? Guidance

Use Arcanara for reflection and decision support, not as a substitute for professional guidance. Treat readings as prompts: "What am I not noticing?" "What tradeoff am I avoiding?" Then make grounded choices with real-world information.

Troubleshooting
"Interaction Failed" or Arcanara doesn't respond. Why? Troubleshooting

Common causes:

  • Missing channel permissions (Send Messages / Embed Links / Attach Files)
  • Discord API hiccups (transient)
  • Command timing out due to heavy load

If persistent, capture a screenshot and share it in the support server with the command and channel name.

"I searched the deck but found no card named ..." What should I do? Troubleshooting

Try a shorter partial name (e.g., "lovers", "high priestess") or select from the autocomplete list. If a common card name isn't matching, report it in the support server.

Support
How do I report a bug or request a feature? Support

Post in the support server's appropriate channel (support / bugs / suggestions). For bugs, include what you ran, what you expected, and what happened. Screenshots are gold.

Support inquiries receive a response within 24 hours or less.
